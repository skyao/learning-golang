### Operators and punctuation/运算符和标点符号

https://golang.org/ref/spec#Operators_and_punctuation

以下字符序列代表运算符（包括赋值运算符）和标点符号。

```go
+    &     +=    &=     &&    ==    !=    (    )
-    |     -=    |=     ||    <     <=    [    ]
*    ^     *=    ^=     <-    >     >=    {    }
/    <<    /=    <<=    ++    =     :=    ,    ;
%    >>    %=    >>=    --    !     ...   .    :
     &^          &^=
```
