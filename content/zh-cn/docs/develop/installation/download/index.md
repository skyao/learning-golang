## 下载

下载地址： 

https://go.dev/dl/

windows 选择 msi 安装文件:

https://go.dev/dl/go1.24.2.windows-amd64.msi

linux 下选择 tar.gz 压缩文件:

```bash
wget https://go.dev/dl/go1.24.2.linux-amd64.tar.gz
```

版本一般选最新。

## 准备gopath

gopath 是 golang 的 workspace，用于存放项目代码和依赖。

```bash
mkdir -p $HOME/work/soft/gopath/
```
